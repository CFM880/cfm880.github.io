1.1 这一章讲述了数据库系统的几个主要优点。它有哪两点不足之处
1.解决数据的冗余和不一致
2.数据访问更容易
3.数据与数据之间不独立
4.一致性约束
5.原子访问
6.并发访问
7.更安全
缺点：多层次依赖

1.2 列出Java或C++之类的语言中类型说明系统与数据库系统中使用的数据定义语言的5个不同之处

1. Java以类表示一个类别，数据库系统用表来表示
2. Java中变量的修改添加删除都是在内存中操作，数据库系统是在磁盘上操作
3. Java中一个类型的属性是不可动态添加的，数据库系统可以
4. Java是过程化的，数据库系统语言是非过程化
